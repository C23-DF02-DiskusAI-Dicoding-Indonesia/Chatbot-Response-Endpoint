# Chatbot-Response-Endpoint


## Run Locally

Clone the project

```bash
  git clone https://github.com/C23-DF02-DiskusAI-Dicoding-Indonesia/Chatbot-Response-Endpoint.git
```

Go to the project directory

```bash
  cd chatbot
```

Install dependencies

```bash
  pip Install -r requirement.txt
```

Start the server

```bash
  python app.py
```
